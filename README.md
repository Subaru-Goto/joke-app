# Joke app

This app is to show random jokes.

### Prerequisites

- Node.js
- npm

### How to start

1. Clone the repo: `git clone https://github.com/yourusername/your-repo.git`
2. Install NPM packages: `npm install`
3. Start the server: `npm start`
